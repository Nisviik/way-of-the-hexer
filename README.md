# Way of the Hexer
Foundry VTT implementation of [Way of the Hexer](https://www.pathfinderinfinite.com/en/product/565589/way-of-the-hexer?affiliate_id=2327748&src=github) book by nisviik.


## Installation
In Foundry VTT, go to the **Add-on Modules** tab, click **Install Module**, and paste the following into the **Manifest URL** field:
```text
https://github.com/nisviik/way-of-the-hexer/releases/latest/download/module.json
